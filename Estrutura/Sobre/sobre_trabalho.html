Sobre o Trabalho

Problemática

Apresentação do setor

Tema: comércio local - Reúne pequenos negócios que movimentam a economia de comunidades, oferecendo produtos e serviços diretamente à população. Esse setor valoriza a cultura regional e proporciona atendimento mais próximo ao cliente, mas enfrenta desafios para se adaptar ao ambiente digital diante da concorrência com grandes plataformas online.

Problema do setor

Pequenos restaurantes têm dificuldade em divulgar seus produtos e atender pedidos online de forma eficiente. A dependência de aplicativos de delivery, que cobram altas taxas, reduz o lucro e limita a autonomia desses comércios no contato direto com seus clientes.

Qual o problema a ser tratado

A falta de uma presença digital própria impede que o restaurante Kishimoto alcançar mais clientes e gerenciar pedidos online sem intermediários. Isso evidencia a necessidade de uma solução acessível para divulgar o cardápio e facilitar as vendas.

Solução Proposta

Desenvolver um site institucional para uma loja de culinária japonesa, com um design atrativo e funcionalidade de vendas online. O site permitirá apresentar a história da empresa, divulgar o cardápio completo e oferecer um carrinho de compras interativo para facilitar pedidos diretos, aproximando o restaurante de seus clientes sem depender de plataformas terceirizadas.

Qual a Proposta de Valor do Site

O site oferece ao restaurante uma plataforma própria para expor seus produtos e atender pedidos online, aumentando o alcance digital e proporcionando mais autonomia no relacionamento com os clientes. Para o consumidor, garante uma experiência prática, visualmente agradável e direta ao comprar seus pratos favoritos.

Tecnologias Utilizadas

Foram utilizados HTML, CSS e JavaScript para a construção do site, além do framework Bootstrap e do Google Fonts para aprimorar o design e a tipografia.
Para o desenvolvimento e execução do projeto, foi utilizada a IDE Visual Studio Code, com o plugin Live Server, necessário para rodar e testar o site localmente.

Explicação Breve Sobre o uso de HTML E CSS e JavaScript

O site foi desenvolvido com HTML, responsável pela estrutura e organização do conteúdo das páginas; CSS para estilização e definição do layout, além do uso do Bootstrap como framework para facilitar a criação de um design responsivo e moderno; e JavaScript, utilizado para adicionar interatividade, como o carrinho de compras funcional. Para execução e testes, foi utilizado o Live Server no Visual Studio Code.

Destaques Tecnicos

O projeto inclui o uso de componentes renderizados por JavaScript, um carrinho de compras funcional que calcula automaticamente o total e permite adicionar ou remover itens, além de elementos interativos como botões e modais. O site também conta com layout proprio com auxilio do Bootstrap, garantindo boa experiência de navegação.

Quais Funcionalidades em JavaScript Foram Incluídas?

Foram implementadas funcionalidades como:
- Carrinho de compras: permite adicionar, remover e visualizar produtos com cálculo automático do valor total.
- Renderização dinâmica de componentes: os elementos da tela, como o Header, Footer e o Carrinho são carregados e exibidos via JavaScript.
- Interações visuais: atualização em tempo real do carrinho conforme o usuário interage e seu formato, em modal.
